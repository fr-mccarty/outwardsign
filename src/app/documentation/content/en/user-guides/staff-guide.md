# Staff Guide

This guide is for parish staff members who plan and execute sacramental celebrations.

## Your Role as Staff

As a staff member, you can:

- Create and manage all sacramental events (weddings, funerals, baptisms, etc.)
- Assign ministers and roles to liturgical celebrations
- Edit petition templates and readings
- Generate printed liturgical scripts
- Invite parishioners to view their events
- Access the full parish directory

## Common Tasks

### Planning a Wedding

1. Navigate to **Weddings** in the sidebar
2. Click **Create Wedding**
3. Add the bride and groom (or create new people)
4. Set the wedding event date and location
5. Choose readings and petitions
6. Assign ministers (presider, lectors, musicians)
7. Print the ceremony script

### Scheduling a Mass

1. Navigate to **Masses** in the sidebar
2. Click **Create Mass**
3. Link to the liturgical calendar event (if applicable)
4. Select the presider and homilist
5. Add petitions and announcements
6. Assign liturgical roles (lectors, EMHCs, servers)
7. Generate the Mass script

### Managing People

1. Navigate to **People** in the sidebar
2. Use the search to find existing parishioners
3. Click **Create Person** to add new directory entries
4. Assign people to groups and roles
5. View a person's sacramental history

## Best Practices

### Before the Celebration

- **Confirm Ministers** - Check that all assigned ministers have confirmed
- **Review Readings** - Ensure readings are appropriate for the liturgical season
- **Print Scripts** - Have scripts printed and in the sacristy 24 hours before
- **Communicate** - Send event details to all participants

### During Planning

- **Use Templates** - Start with petition templates and customize as needed
- **Check the Calendar** - Avoid scheduling conflicts
- **Assign Roles Early** - Give ministers adequate notice
- **Save Often** - Changes are saved automatically, but review before finalizing

### After the Celebration

- **Update Status** - Mark events as "Completed"
- **Archive Scripts** - Keep digital copies of all printed scripts
- **Follow Up** - Send thank you notes or feedback requests
- **Review** - Note what worked well for future planning

## Keyboard Shortcuts

- `⌘ + K` - Open search
- `⌘ + P` - Print current page
- `⌘ + S` - Save form (when editing)
